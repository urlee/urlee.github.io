【重要なお知らせ】インターネットセキュリティに関する設定確認のお願い

先日インターネット通信で使用する暗号化方式「SSL3.0」において脆弱性が発見され、暗号化された内容が漏洩する可能性があることが発表されました。 弊社におきましては、この脆弱性への対策を完了しております。

対策の影響により、ご利用環境よっては設定をご変更いただきませんと、今後ログインが出来ない事象が生じます。 大変お手数ではございますが、以下をご確認のうえ、設定の変更をお願い致します。

ご利用環境

当サイトで快適にお買い物をしていただくために、お使いのブラウザの環境設定をしていただくことをおすすめいたします。

推奨ブラウザ

Windowsをお使いの場合

  • Microsoft Internet Explorer 最新版
  • Mozilla FireFox 最新版
  • Google Chrome 最新版

推奨する設定

  • JavaScriptを有効にする
  • Cookie利用を有効にする
  • スタイルシートを有効にする
  • SSLを有効にする

【Internet Explorer6をご利用の皆様へ】

ログイン時やマイページ閲覧時にセキュリティエラーが出る場合、以下のセキュリティ設定をしてご利用ください。

  1. インターネットオプションを表示させます Internet Explorerを開き、「ツール」→「インターネットオプション」をクリックします
  2. 「詳細設定」タブをクリックし、「セキュリティ」ゾーンを選びます。
  3. 「SSL 3.0 を使用する」チェックボックスをオフにし、「TLS 1.0 を使用する」、「TLS 1.1 の使用」、
    および「TLS 1.2 の使用」チェック ボックスをオンにします (使用可能な場合)。
  4. OKをクリックします。
  5. Internet Explorerを再起動します。

マイクロソフトHP：https://technet.microsoft.com/ja-jp/library/security/3009008.aspx
